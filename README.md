# 🏬 Retail Store Performance Comparator (Streamlit Project)

## 📌 Overview
This is an interactive dashboard built using **Streamlit** to compare performance between two selected retail stores across selected KPIs and months.

The dashboard enables dynamic filtering and visually highlights the top-performing store along with the quantified performance gap to support data-driven business decisions.

---

## 🚀 Features
- Compare two stores side-by-side
- KPI selection (Sales, Profit, Transactions)
- Multi-month filtering
- Executive-style winner insight
- Dynamic grouped bar chart comparison
- Clean two-column dashboard layout

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- NumPy

---

## 📊 Dashboard Layout

Left Section:
- KPI Summary
- Store 1 vs Store 2 comparison
- Winner & Lead insight

Right Section:
- Monthly grouped bar chart comparison

---

## ▶️ How to Run Locally

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run store_performance_comparison.py
